# Nexus Warehouse System

Complete warehousing solution with an intelligent network, interactive globe, and 3PL management.

## Project Structure
- **/nexusvc-main**: React Frontend (Vite)
- **/warehouse-backend**: Node.js Express API
- **/database**: PostgreSQL Schema & Migrations

## Hosting Strategy
- **Frontend**: Render (Static Site)
- **Backend**: Render (Web Service)
- **Database**: Render (PostgreSQL)
